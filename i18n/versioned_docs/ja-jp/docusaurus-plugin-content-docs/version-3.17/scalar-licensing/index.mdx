# ライセンスキーの設定方法

import TranslationBanner from '/src/components/_translation-ja-jp.mdx';

<TranslationBanner />

ScalarDB Enterprise Standard/Premium または ScalarDB Analytics を実行するには、ライセンスキーを設定する必要があります。

- ScalarDB Enterprise Standard/Premium または ScalarDB Analytics の商用ライセンスキーをお持ちの場合は、[商用ライセンスキーの設定方法](./commercial.mdx)を参照してライセンスキーを設定してください。
- ScalarDB Enterprise Standard/Premium または ScalarDB Analytics を評価したい場合は、[試用ライセンスキーの設定方法](./trial.mdx)で提供されている試用ライセンスキーを使用できます。
