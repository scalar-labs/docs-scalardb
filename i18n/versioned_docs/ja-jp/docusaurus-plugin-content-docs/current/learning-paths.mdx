# ScalarDB 学習パス

import TranslationBanner from '/src/components/_translation-ja-jp.mdx';

<TranslationBanner />

このガイドでは、さまざまな役割に応じた学習パスを提供します。あなたの役割に応じて、適切なドキュメント順序に従って、ScalarDB の包括的な理解を得てください。

:::tip

ScalarDB に関する情報は、本サイト上のドキュメントと、スライド形式の[オンボーディングコンテンツ](./onboarding.mdx)で提供しています。本サイト上のドキュメントは、実際の利用方法に加え、機能、設定、運用に関する情報を網羅的に掲載した包括的なドキュメントです。一方、オンボーディングコンテンツは、ScalarDB を理解し始めるために重要な情報を厳選してまとめた入門用コンテンツです。まず全体像を素早く把握したい場合はオンボーディングコンテンツを、実際の利用方法や詳細な仕様、設定・運用情報を確認したい場合は本サイトを参照してください。

:::
