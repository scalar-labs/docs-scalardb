このページでは、ScalarDB のエラーコードの一覧を示します。

## エラーコードのクラスと説明

| クラス        | 説明                                           |
|:-------------|:----------------------------------------------|
| `CORE-1xxxx` | ユーザーエラーカテゴリのエラー                     |
| `CORE-2xxxx` | 同時実行エラーカテゴリのエラー                     |
| `CORE-3xxxx` | 内部エラーカテゴリのエラー                        |
| `CORE-4xxxx` | 不明なトランザクションステータスエラーカテゴリのエラー |
