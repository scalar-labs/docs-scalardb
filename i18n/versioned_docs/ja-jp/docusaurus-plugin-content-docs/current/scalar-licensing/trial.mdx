# 試用ライセンスキーの設定方法

import TranslationBanner from '/src/components/_translation-ja-jp.mdx';

<TranslationBanner />

ScalarDB Enterprise Standard/Premium および ScalarDB Analytics 用の以下の試用ライセンスキーを使用できます。商用ライセンスキーをお持ちの場合は、[商用ライセンスキーの設定方法](./commercial.mdx)を参照してライセンスキーを設定してください。

ScalarDB Enterprise Standard/Premium または ScalarDB Analytics を実行するには、`.properties` ファイルを作成し、試用ライセンスキーと証明書をファイルに追加する必要があります。`.properties` ファイルには、使用している製品に基づいて以下の設定をコピーし、ファイルに貼り付けてください。

:::warning

- これらのライセンスキーは評価目的でのみ有効です。
- These licensing keys are valid for evaluation purposes only.
- 試用ライセンスを使用する場合、ScalarDB Cluster および ScalarDB Analytics はライセンスの検証と有効期限の確認のためにインターネットに接続されている必要があります。
- 試用ライセンスキーは定期的に更新されます。本番環境での使用については、[お問い合わせ](https://www.scalar-labs.com/contact)して商用ライセンスを取得してください。

:::

:::note

ScalarDB Core は、[GitHub](https://github.com/scalar-labs/scalardb) で Apache 2.0 ライセンスの下でオープンソースソフトウェアとして利用可能です。

:::
