# 商用ライセンスキーの設定方法

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';
import TranslationBanner from '/src/components/_translation-ja-jp.mdx';

<TranslationBanner />

ScalarDB Enterprise Standard/Premium または ScalarDB Analytics を実行するには、`.properties` ファイルを作成し、商用ライセンスキーと証明書をそのファイルに追加する必要があります。`.properties` ファイルで、使用している製品に基づいて次の設定のいずれかをコピーし、その内容を `.properties` ファイルに貼り付けて、`<YOUR_LICENSE_KEY>` をライセンスキーに置き換えます。

:::note

- ライセンスキーをお持ちでない場合は、すぐに使用できるキーについては[試用ライセンスキーの設定方法](./trial.mdx)をご覧いただくか、[お問い合わせ](https://www.scalar-labs.com/contact)して商用ライセンスを取得してください。
- ScalarDB のライセンス機構は、ライセンスキーの署名、有効期限、およびエディションを検証します。ただし、実行中の Pod の数や契約固有の制限などのデプロイメントの詳細は確認しません。そのため、技術的には契約で許可されている数以上の Pod をデプロイできる場合がありますが、契約で許可されている数の Pod のみを使用するようにしてください。

:::
