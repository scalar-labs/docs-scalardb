# ScalarDB SQL エラーコード

このページでは、ScalarDB SQL のエラーコードの一覧を示します。

## エラーコードのクラスと説明

| クラス       | 説明                       |
|:------------|:--------------------------|
| `SQL-1xxxx` | ユーザーエラーカテゴリのエラー |
