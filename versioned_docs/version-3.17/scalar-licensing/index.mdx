# How to Configure a License Key

To run ScalarDB Enterprise Standard/Premium or ScalarDB Analytics, you need to configure a license key.

- If you have a commercial license key for ScalarDB Enterprise Standard/Premium edition or ScalarDB Analytics, please refer to [How to Configure a Commercial License Key](./commercial.mdx) to configure your license key.
- If you want to evaluate ScalarDB Enterprise Standard/Premium edition or ScalarDB Analytics, you can use the trial license keys provided in [How to Configure a Trial License Key](./trial.mdx).
